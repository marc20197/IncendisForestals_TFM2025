<h1>Treball final del màster de Data Science</h1>
<h2>Descripció</h2>
<p>Aquest projecte correspon a la creació d'una visualització en la qual es pot veure informació sobre els incendis forestals que hi ha hagut a Catalunya en els últims 20 anys i informació meteorològica recollida en aquests 20 anys.
S'ha documentat aquest procés en una memòria.</p>
<p>En concret, la visualització ha estat creada utilitzant l'eina Shiny de R, i després a estat publicada a shinyapps.
  Els fitxers importants han estat dos fitxers utilitzats per preparar les dades (un notebook de Python i un notebook de R) i el codi de l'aplicació (en R), utilitzat per crear la visualització. Es descriurà on trobar aquests fitxers i la visualització a continuació.</p>

<h2>Estructura del GIT</h2>
Els dos fitxers utilitzats per preparar les dades han estat:
<ul>
  <li>Preparació_dades_incendis.Rmd, que correspon a un notebook de R utilitzat per preparar dades sobre els incendis forestals</li>
  <li>preparacio_dades_meteorologiques.ipynb, que correspon a un notebook de Python utilitzat per preparar dades meteorològiques</li>
</ul>
Tots dos fitxers es poden trobar a la carpeta principal d'aquest Git. A part d'això, també trobem el fitxer Enllaços.txt, aquest fitxer és important perquè en ell s'hi poden trobar els
enllaços necessàris per a veure la visualització i el codi necessari per crear-la. Aquests enllaços són:
<ul>
  <li>L'enllaç al repositori de GitHub on ens trobem</li>
  <li>L'enllaç on es pot veure la visualització publicada a Internet</li>
  <li>L'enllaç on podem trobar el codi que ha estat programat per crear la visualització i també la carpeta on el codi va a buscar els 4 fitxers necessaris per carregar les dades.
  Aquest és un enllaç de Google Drive. El motiu de no haver-ho guardat en aquest repositori de GitHub és que alguns dels fitxers  necessàris per carregar dades pesen més de 25Mb i GitHub no permet pujar-los
  En aquest enllaç hi ha tot el necessari per executar la visualització.</li>
</ul>
